# HSK 3 Study Game
### Tkinter based desktop application to effectively memorize HSK 3 vocabolary

---

## Getting started
`pip3 install -r requirements.txt`

#### Study with pinyin
`python3 main_py.py`

#### With pinyin
`python3 main_no_py.py`
